07 — Project Execution

Phased Build Plan

Four phases allow the project to be budgeted over time. Each phase is independently completeable — no phase leaves the yard in a bad state.

Phase 1 — Foundation
Demo, Cleanup & Drainage
Remove hot tub, railroad ties, dead tree, and all debris. Haul off-site. Grade and level yard. Address drainage at driveway-adjacent wall — install French drain or deco drain channel. Repaint all CMU block walls. Install side gate for motorcycle and trash access. Power-wash and reseal existing patio. Repot pomegranate into large decorative planter.
Est. Cost: $8,000 – $18,000
Duration: 2–3 weeks · Crew: Demo + landscape
Phase 2 — Core Structures
Gazebo + Patio Extension + Turf
Extend existing patio with pavers (travertine or stamped concrete). Install louvered metal gazebo (client referenced motorized Costco-style — opens/closes for rain and sun). Install synthetic turf in kids' play zone with proper weed barrier and drainage base. Lay DG landscape and accent rock in remaining yard areas.
Est. Cost: $22,000 – $40,000
Duration: 3–5 weeks · Crew: Concrete, masonry, landscape
Phase 3 — Outdoor Kitchen + Lighting
Kitchen Island, BBQ & Landscape Lighting
Run gas line and 240V electric to outdoor kitchen zone off the back door area. Install outdoor kitchen island: built-in grill, refrigerator, counter. Install overhead string lights at patio, LED uplights on trees and CMU walls, low-voltage path lights along turf border. Install free-standing swing structure in corner zone near existing trees.
Est. Cost: $18,000 – $38,000
Duration: 3–4 weeks · Crew: Electrician, gas plumber, landscape
Phase 4 — Future (Optional)
Casita + Premium Upgrades
Client mentioned desire to add small casita (ADU) in rear portion of yard. Phase 4 addresses: utility stub-outs for casita, final landscaping of rear half of yard, premium smart lighting system, motorized screens on gazebo/pergola, and outdoor sound system integration.

Note: HOA + city permitting required before casita build. We can assist with design documentation for permit applications.
Est. Cost: $30,000 – $80,000+
Duration: 4–8 weeks · Crew: General contractor + sub-trades

⚡ Utility & Infrastructure Notes

Gas and electric discussed to run from the house, coming right off the back door. The existing patio was likely poured to allow future connections — check for existing conduit or stub-outs before trenching to save cost.

A disconnected fish pond electrical connection exists in the yard — can potentially be rerouted to serve the outdoor kitchen zone rather than trenching new conduit.

A drainage slot/hole is visible near the driveway-adjacent wall. This is the original drainage path. Any new hardscape must maintain or improve this drainage — French drain or deco drain channel recommended in Phase 1.
